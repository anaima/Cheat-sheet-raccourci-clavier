Les raccourcis clavier sur Linux :

  • Alt + F4 = Fermer la fenêtre actuelle.
  • Alt + Tab = Permet de changer de fenêtre rapidement
  • Ctrl + Touche Windows + D = Cache toutes les fenêtres et affiche le bureau. Appuyez à nouveau sur les touches pour restaurer vos fenêtres.
  • Ctrl + Alt + L = Verrouiller l'écran.
  • Ctrl + (+ ou -) = zoom + ou zoom -
  • Ctrl + A = Sélectionner tout le texte ou tous les éléments
  • Ctrl + C = Copier le texte ou les éléments sélectionnés
  • Ctrl + V = Coller le contenu
  • Ctrl + Z = Annuler la dernière action
  • Ctrl + Alt + Flèche haut ou flèche bas = permet de basculer sur un autre espace de travail
  • touche Windows + Maj + s = Permet de visualiser vos espaces de travail
  • Ctrl + Alt + t = Permet d'ouvrir le terminal (console)
